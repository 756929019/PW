# PW

#### 介绍
wpf

#### 软件架构
软件架构说明
Visual Studio 2015
.NET Framework 4.6

Prism.Core 7.0
Prism.Mef 6.3
Prism.Wpf 6.3

EntityFramework 6.2
MySql.Data 6.10.9

#### 安装教程

1.  初始化NuGet包
2.  安装EF需要安装文件夹下程序
3.  单独生成各模块（Aside、Chat、Footer、SystemHeader、Map、SystemSet、Tools）
4.  运行PW.Desktop

数据库暂无
